Parameters
rendererThe renderer to be used.
ttfFontPath(optional) The path to the TrueType font to be used by the Renderer, including its filename. It defaults to the font provided by the engine but, even if the same one is used for an game, most probably the path to where it is placed will need to be specified here. Only one font can be used by the engine at present but that will change in the near future.
size(optional) the size of the font in the created images. It defaults to 48.
